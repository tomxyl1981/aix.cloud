开发文档

快速接入 AIX Cloud API

🚀 快速开始

AIX Cloud 提供与 OpenAI 兼容的 API 接口，只需修改 base_url 即可使用。

安装 SDK

pip install aixcloud

Python 示例

from aixcloud import AIX client = AIX( api_key="sk-aix-xxxx", base_url="https://api.aix.cloud/v1" ) response = client.chat.completions.create( model="AIX-Tech", messages=[{role: "user", content: "你好"}] ) print(response.choices[0].message.content)

🔐 认证

使用 API Key 进行认证，请求头格式：

Authorization: AIX-Cloud YOUR_API_KEY

curl 示例：

curl -X POST https://api.aix.cloud/v1/chat/completions \ -H "Authorization: AIX-Cloud sk-aix-xxxx" \ -H "Content-Type: application/json" \ -d '{"model": "AIX-Tech", "messages": [{"role": "user", "content": "Hello"}]}'

💬 聊天 API

请求地址

POST https://api.aix.cloud/v1/chat/completions

请求参数

  • model - 模型名称，目前仅支持 AIX-Tech
  • messages - 消息数组，格式为 [{"role": "user", "content": "内容"}]
  • temperature - 生成随机性（0-2，默认 0.7）
  • max_tokens - 最大生成 token 数
  • stream - 是否流式返回（true/false）

🌊 流式响应

from aixcloud import AIX client = AIX( api_key="sk-aix-xxxx", base_url="https://api.aix.cloud/v1" ) stream = client.chat.completions.create( model="AIX-Tech", messages=[{role: "user", content: "你好"}], stream=True ) for chunk in stream: if chunk.choices[0].delta.content: print(chunk.choices[0].delta.content, end="")

🤖 支持的模型

当前支持的模型：

  • AIX-Tech - AIX Cloud 智能模型

获取模型列表：

GET https://api.aix.cloud/v1/models

⏱️ 调用限制

  • 速率限制：10 次请求/分钟
  • 每个 API Key 独立计算

⚠️ 错误处理

  • 400 - 请求参数错误
  • 401 - API Key 无效
  • 402 - 余额不足
  • 429 - 请求频率超限
  • 500 - 服务器内部错误
